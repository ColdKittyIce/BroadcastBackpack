🚀 Getting Started

Broadcast Backpack is a desktop application for managing live broadcasts. It provides a soundboard, music queue, recording, streaming, and session logging — all in one window.

Quick Start

  1. Launch the application
  2. Configure your audio output device in Settings → Audio
  3. Add sounds to your soundboard by dragging audio files onto buttons
  4. Add music to the queue by dragging files to the bottom panel
  5. Press GO LIVE when ready to start your show

🖥️ Interface Overview

🔴

Header Bar

GO LIVE button, timers, ON AIR indicator, PANIC button, and tape recorder controls.

🎹

Soundboard

Pinned row + tabbed banks of sound buttons. Drag-and-drop audio files to assign.

📝

Right Panel

Quick folders, websites, notes tabs, bits board, and session log.

🎵

Bottom Bar

Music queue with transport controls and now-playing display.

🎹 Soundboard

Adding Sounds

  • Drag & Drop: Drag audio files directly onto any button
  • Right-click: Open button settings to browse for a file
  • Supported formats: MP3, WAV, OGG, FLAC

Button Settings

Right-click any button to access:

  • Label: Custom button name
  • Color: Button background color
  • Volume: Individual button volume
  • Loop: Enable looping playback
  • FX: Audio effects (pitch, speed, reverb, echo, filters)

Banks

Organize your sounds into banks (tabs). Configure bank layout in Settings → Soundboard.

Board Volume

The BOARD slider controls overall soundboard volume. The Board Gain setting in Settings → Audio provides additional boost (0-12 dB) for quieter sound files.

🎵 Music Queue

Adding Music

  • Drag audio files to the queue panel
  • Use the folder icon to browse
  • Drag folders to add all audio files

Transport Controls

  • ▶ Play / ⏸ Pause: Start or pause current track
  • ⏹ Stop: Stop playback
  • ⏮ Previous / ⏭ Next: Navigate tracks
  • 🔀 Shuffle: Randomize queue order

Queue Volume

The QUEUE slider controls music playback volume independently from the soundboard.

🎙️ Recording

Tape Recorder

The tape recorder in the header captures audio from your selected input device.

  • ⏺ Record: Start recording
  • ⏹ Stop: Stop and save recording
  • ⏸ Pause: Pause recording
💡 Tip: For full show recording, set your input device to "Stereo Mix" or a virtual audio cable that captures all system audio.

Recording Location

Recordings are saved to your configured recordings folder (Settings → Audio). Default: BroadcastBackpack/recordings

📡 Streaming

Broadcast Backpack can stream directly to Icecast-compatible servers.

Setup

  1. Go to Tools menu → Stream Settings
  2. Enter your server details: host, port, mount point
  3. Enter your source credentials
  4. Select the audio device to stream
  5. Click Connect to start streaming

📋 Session Log

The session log automatically records timestamped events during your show:

  • Sound plays with elapsed time
  • Gold moments (press Ctrl+Shift+G)
  • Manual timestamps (press Ctrl+Shift+T)
  • Phone calls (if MicroSIP integration is enabled)

Exporting

Export your session log from File → Export Session Log, or use the Markers export for audio editing software (Adobe Audition CSV, Audacity labels).

⚙️ Settings

Access settings via Edit → Settings or press Ctrl+,

Tabs

  • Show: Show name, episode number, title template
  • Audio: Output/input devices, recordings folder, fade duration, board gain
  • Soundboard: Bank configuration, pinned slots, playback modes
  • Hotkeys: Customize keyboard shortcuts
  • Visual: Color theme, custom colors, button flash settings
  • Integrations: Discord webhook, MicroSIP setup, browser preference
  • About: Version info, help guide, data folder

🔧 Troubleshooting

No Sound Playing

  • Check your audio output device in Settings → Audio
  • Make sure volume sliders (BOARD, QUEUE) aren't at zero
  • Check individual button volume isn't muted

Soundboard Sounds Too Quiet

  • Increase the Board Gain setting in Settings → Audio (0-12 dB boost)
  • Check individual button volumes
  • Normalize your audio files to a consistent level

Recording Not Working

  • Verify input device is set correctly in Settings → Audio
  • For full show recording, use "Stereo Mix" or a virtual audio cable
  • Check the recordings folder exists and is writable

Stream Won't Connect

  • Verify server address, port, and mount point
  • Check source credentials (username is usually "source")
  • Ensure your server allows SOURCE connections
  • Check firewall settings

Data Location

All settings and data are stored in: C:\Users\[Username]\BroadcastBackpack\

  • config.json — All settings
  • sessions/ — Session logs
  • recordings/ — Tape recordings
  • markers/ — Exported markers
